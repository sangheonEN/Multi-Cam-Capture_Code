딥러닝

딥러닝 : 머신러닝 안에 속한 아주 강력한 학습 방법으로 
사전 훈련법 (야! 이건 아무리 생각해도 Deep하다.)

뉴럴 네트워스 -> deep 네트워크! 딥러닝이 발현함.


제프리 힌튼 --> 딥러닝(딥러닝 기법 중에도 CNN AlexNet 사용)을 이용해 ILSVRC 대회에 출전함. 기계 이미지 인식 대회

CNN : 이미지의 특징을 반복적으로 분석 후 사전 훈련을 시킨 다음에 결과를 다시 인공신경망으로 보내서 최종적으로 어떤 이미지가 A 인지 B 인지 판별

깊은 네트워크 + 다층(은닉층) + 역전파 신경망  --> 이미지 인식 대회를 기준으로 딥러닝이 가장 높은 방법론이였다.

추후에 알파고의 학습도 딥러닝을 사용했는데 13개의 은닉층을 가지고 했다.

약 인공지능 : 인간의 일부분을 학습해서 그 부분에 대한 정보를 조합하고 이해하는 정도이다.

강 인공지능 : 모든 분야에서 사람과 동일한 지능 그 이상의 지능을 가진 Level (문제를 실제로 사고하고 해결할 수 있는 수준 지각 및 자아가 됨)

초 인공지능 : 기계가 모든 면에서 인간의 능력보다 초월한 단계.

기존의 기계학습법과의 가장 큰 차이점은 딥러닝은 데이터 표현법을 학습하는 방법에서 나온다. 

기존 기계 학습은 이미지나 어떤 종류의 데이터가 들어오던 간에 컴퓨터가 이해할 수 있는 벡터나 행렬 형태로 사람이 만들어주면 거기서 부터 학습이 시작됨.
딥러닝 학습은 자기가 알아서 그 데이터로부터 어떤 것을 중요하게 봐야 하는지 알아 채서 데이터를 표현하는 방법까지 자동으로 학습이 진행 됨.


컴퓨터 비전

이미지나 비디오에서 얻을 수 있는 내용(정보)들을 자동으로 얻을 수 있게 해줌.

딥러닝과 연계를 한다면 AI에게 가상의 눈을 붙여주어 정보를 얻을 수 있게 하는 것이다 라고 생각해 볼 수 있다.


딥러닝 + 컴퓨터 비전

딥러닝의 데이터 표현법을 학습하는 방법

CNN

이미지와 같이 2차원 데이터를 공간 자료가 있는 데이터를 이용해 사용함. 

입력데이터(INPUT 이미지와 같은 2차원 데이터)에 필터(Convolution)를 적용해서 변환을 여러번 시키는 것

딥러닝+컴퓨터 비전 응용

1. yolo : 이미지를 7*7 바둑판으로 나눔. 각각의 block이 중심이 자기에 있으면서 물체가 있을 것 같은 박스(바운딩 박스)들을 예측한다. 
사용 알고리즘
각 블록들의 물체가 있으면 어떤 물체가 있는지 구별하기 위해 서로 유사한 물체를 표현하는 블록들은 같은 색으로 표시할 수 있다.

2. 의미론적 이미지 분할 : 이미지 안의 픽셀이 어떤 오브젝트(클래스)에 속하는지 결정해 주는 것이다. ex) 자율주행할때 차량, 인간, 표지판, 나무 등 도로에서 볼 수 있는 객체들을 의미론적인 특징 정보를 통해 각 물체를 식별하는 것이다.
사용 알고리즘
encoder에서는 입력데이터(이미지)에 해당 되는 특징을 추출해서 서로 유사한 클래스로 구분하고 
dencoder에서는 추출된 정보를 바탕으로 그 픽셀이 어떤 물체인지 클래스를 예측함.

3. 물체 추적 : 내가 원하는 물체를 찾아서 보여주는 방법
사용 알고리즘
추적 대상 후보를 샘플링 -> 추적 대상 발견 -> 학습 데이터 수집 -> CNN 업데이트 (다음 프레임 마다 계속 이러한 과정 반복)

4. 이미지와 언어데이터를 같이 활용한 방법
사용 알고리즘
이미지 정보 + 언어 데이터의 두 가지의 입력데이터가 활용 된다. -> 이미지입력데이터에서 특징을 추출하는 encoder(CNN사용) + 언어입력데이터에서 특징을 추출하는 encoder(RNN사용) 을 사용해서 특징을 추출하고 분류기를 통해 원하는 답을 예측해낸다.
4-1 이미지 내용 기술(Image Captioning)이미지를 묘사하는 언어데이터를 출력하는 방법

5. 이미지 생성 신경망 (GAN 적대적 생성 신경망)
